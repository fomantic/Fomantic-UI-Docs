Reset
A reset is a set of normalized values for CSS properties that correct for abberations in browser defaults

Overview Test

What's In Our Reset

Fomantic's default theme includes the latest Normalize CSS to provide a base line HTML reset. In addition, Fomantic UI requires a Box-sizing reset, to make sure that elements handle width definitions in the same way.

The reset is required for Fomantic UI components to function properly, and should also be included when using individual components.

Reset Options

The basic themes only includes the required box-sizing reset and nothing else. The resetcss theme provides a version of Reset CSS, a less opinionated CSS reset.
